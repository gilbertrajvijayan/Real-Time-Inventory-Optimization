# Real-Time Inventory Optimization Using Predictive Analytics and IoT

## About
A framework for real-time inventory optimization combining IoT sensor 
data with predictive analytics to automate replenishment decisions 
and reduce stockouts.

- 📄 Inventory_optimization_report.docx — Full project report
- 📊 Presentation.pptx — Project presentation slides

## Tech Used
- Python (Pandas, NumPy, PySpark, Statsmodels)
- Machine Learning (ARIMA, SARIMA, XGBoost, K-Means)
- Tableau / Power BI
- SQL & IoT Integration
